# Personal Portfolio

My personal developer portfolio showcasing my projects, technical skills, and experience in backend development, machine learning, and quantitative finance.

## Live Website

https://your-portfolio-link.vercel.app/

## Features

- Responsive design
- Project showcase
- Experience & Education timeline
- Resume download
- Social links
- Smooth scrolling animations

## Built With

- React
- Vite
- Tailwind CSS
- JavaScript

## Run Locally

```bash
git clone https://github.com/abhas-singh15/portfolio.git

cd portfolio

npm install

npm run dev
```
